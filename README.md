bash main.sh <arg1> ... <arg7>.
*les arguments sont tels que :*.
** -d1 ; -d2 ; -l ; -t ; -s ; -h  bibliographie.
 -d1 : _traitement d1_ .
 -d2 : _traitement d2_ .
  -l : _traitement l_.
 -t : _traitement t_ .
 -s : _traitement s_ .
 -h : _montrer l'aide (prioritaire)_. 
 -bibliographie : _liste des sources_ **.
